# Intelligent AI Delegation

**Tomašev, Franklin, Osindero (Google DeepMind, 2026)**

AI エージェントによるタスク委任（delegation）の体系的フレームワークを提案する position paper。既存の multi-agent delegation がヒューリスティックに依存し動的適応や障害回復が不十分であるとの問題意識から、5つの柱（Dynamic Assessment, Adaptive Execution, Structural Transparency, Scalable Market Coordination, Systemic Resilience）に基づく9つの技術的プロトコル（Task Decomposition, Task Assignment, Multi-objective Optimization, Adaptive Coordination, Monitoring, Trust & Reputation, Permission Handling, Verifiable Task Completion, Security）を定義する。人間組織論（Principal-Agent Problem, Span of Control, Authority Gradient）からの知見を AI delegation に適用し、既存プロトコル（MCP, A2A, AP2, UCP）の gap analysis と具体的な拡張案を提示する。実験的検証は含まれず、conceptual framework / research roadmap として位置づけられる。
