# Intelligent AI Delegation

**Authors:** Nenad Tomašev, Matija Franklin, Simon Osindero
**Affiliation:** Google DeepMind
**Year:** 2026
**Type:** Position Paper / Conceptual Framework

AI agent がますます複雑なタスクを遂行可能になる中、著者らは既存の multi-agent delegation が heuristic に依存し、動的適応・障害回復・説明責任が不十分であると主張する。本論文は「intelligent delegation」を、権限の移譲・責任・説明責任・信頼確立・検証可能性を含む一連の意思決定プロセスとして定義し、5つの要件 pillar（Dynamic Assessment, Adaptive Execution, Structural Transparency, Scalable Market Coordination, Systemic Resilience）と9つの技術的プロトコル（Task Decomposition, Task Assignment, Multi-objective Optimization, Adaptive Coordination, Monitoring, Trust & Reputation, Permission Handling, Verifiable Task Completion, Security）から成る包括的フレームワークを提案する。人間組織論（Principal-Agent Problem, Span of Control, Authority Gradient, Zone of Indifference, Transaction Cost Economics）の知見を AI delegation に適用し、既存プロトコル（MCP, A2A, AP2, UCP）の gap analysis と具体的拡張案を提示する。実験的検証は含まれず、conceptual framework / research roadmap として位置づけられる。
